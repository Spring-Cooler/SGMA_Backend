## 프로젝트 개요 <a id="프로젝트-개요"></a>
---
### 1. 배경 <a id="1-배경"></a>
1. 팀 스터디의 효율성을 향상시킬 수 있는 도구가 필요함
2. 일정 관리, 학습 진도 파악 등 스터디의 정보를 한 곳에서 모아볼 수 있는 서비스가 없음
### 2. 서비스 목표 <a id="2-서비스-목표"></a>
> 팀 스터디의 모든 과정을 쉽게 관리 및 각 멤버가 자신의 학습 목표를 달성할 수 있도록 도움
